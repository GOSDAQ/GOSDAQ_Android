# GOSDAQ_Android
주식 포트폴리오 관리 서비스, GOSDAQ

# Envrionment
* macOS 13.1 (Ventura)
* Kotlin
* MVP 패턴
* Room Database
* Retrofit2
* Hilt

# How to run
* 디렉토리 최상단에 `local.properties` 파일을 추가합니다.
* 추가한 `local.properties` 내용에 `api.url='API Server URL'`을 추가합니다.
* 이후 gradle 명령어를 활용하여 어플리케이션을 빌드합니다.
